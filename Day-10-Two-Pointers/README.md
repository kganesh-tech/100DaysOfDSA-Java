1. Reverse a String
**Difficulty:** Easy
**Pattern:** Two Pointers

 **Problem Statement**
Given a character array, reverse the array in-place.

**Approach:**
Use two pointers: one at the beginning and one at the end
Swap elements at both pointers
Move pointers toward each other until they meet

**Complexity:**
Time Complexity: O(n)
Space Complexity: O(1)

2. Reverse Vowels of a String
**Difficulty:** Easy
**Pattern:** Two Pointers

 **Problem Statement**
Given a string, reverse only the vowels in it and return the result.

**Approach**:
Convert string to character array
Use two pointers (start & end)
Move pointers until vowels are found
Swap vowels and continue
 
 **Complexity:**
Time Complexity: O(n)
Space Complexity: O(n) (due to char array conversion)
